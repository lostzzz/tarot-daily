# 塔罗牌占卜网页 - 项目规格

## 1. 项目概述
- **项目名称**: Tarot Daily - 每日塔罗占卜
- **类型**: 单页 Web 应用（手机优先）
- **核心功能**: 用户每日随机抽取一张塔罗牌，获得占卜结果
- **目标用户**: 对塔罗牌感兴趣的用户

## 2. UI/UX 规格

### 布局结构
- **移动优先设计**（最小宽度 320px）
- 单页应用，无刷新
- 底部固定按钮区域

### 页面结构
1. **首页** - 品牌展示 + 抽取按钮
2. **结果页** - 展示抽到的牌 + 牌义解释 + 每日运势

### 视觉设计
- **主色调**: 
  - 背景: #0a0a0f (深邃夜空黑)
  - 主色: #9b59b6 (神秘紫)
  - 强调色: #f1c40f (金色)
  - 文字: #ecf0f1 (月光白)
- **字体**: 
  - 标题: 'Cinzel', serif (神秘感)
  - 正文: 'Noto Sans SC', sans-serif (中文)
- **效果**: 
  - 卡片悬停发光效果
  - 抽取动画（旋转 + 放大）
  - 神秘星光背景动画

### 组件
- 塔罗牌卡片（带正反面）
- 抽取按钮（带脉动动画）
- 牌义解释卡片
- 分享按钮

## 3. 功能规格

### 塔罗牌数据
- 78张牌（大阿尔卡纳22张 + 小阿尔卡纳56张）
- 每张牌包含：
  - 名称（中文 + 英文）
  - 牌面图像（使用 Unicode 符号或CSS绘制）
  - 正位含义
  - 逆位含义
  - 每日建议

### 核心逻辑
1. 首次访问 → 引导抽取
2. 每日一次 → 本地存储记录（localStorage + ontology）
3. 已抽取 → 显示今日结果 + 倒计时

### 数据存储
- localStorage: 当日抽取记录
- ontology: 长期历史记录

## 4. 部署
- 目标: 可外网访问
- 方案: 使用 Node.js 服务或静态托管

- [ ] 抽取动画流畅

- [ ] 每日限制生效
